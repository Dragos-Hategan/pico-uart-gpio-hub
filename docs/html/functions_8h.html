Detailed Description

Utility functions for UART, LED control, and GPIO pin parsing.

Provides helpers to:

  • Initialize and blink the onboard LED
  • Configure UART with specific TX/RX pins
  • Receive UART data into buffers
  • Parse UART messages for TX/RX pin pairs
  • Reset GPIO pins to default SIO mode

void uart_init_with_pins ( uart_inst_t * uart,
uart_pin_pair_t pin_pair,
uint32_t baud_rate )

Initializes UART and configures TX/RX GPIO pins.

  • Deinitializes the UART first to reset state.
  • Assigns UART function to the specified TX and RX pins.
  • Initializes the UART with the provided baud rate.
  • Waits 1 ms for stability.
Parameters
uartUART instance (e.g., uart0 or uart1).
pin_pairThe TX/RX pin pair to configure.
baud_rateUART baud rate (e.g., 9600, 115200).
